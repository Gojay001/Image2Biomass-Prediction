# 数据划分约定（按图像）

本赛题 **`train.csv` 为「一行一个 (图像, target)」**：同一 `image_path` 有多行（5 个 `target_name`）。若随机对**行**做 train/val 划分，会把**同一图像**同时放进训练与验证，造成 **标签泄漏**（验证分数虚高）。

---

## 1. 推荐做法

1. 从 `train.csv` 提取 **唯一图像键**：如 `image_path` 或 `sample_id` 中 `__` 前的图像 ID 部分。
2. 在 **唯一图像集合** 上做 `GroupKFold` / `train_test_split(..., groups=image_id)` 等，保证 **同一图像只出现在 train 或 val 一侧**。
3. 划分后再展开回「行」级样本用于训练（每张图仍对应 5 行标签）。

---

## 2. 加权指标

- 官方榜为 **加权 R²**（见 `docs/competition-spec.md`）。本地验证可：
  - 实现与官网一致的 **R²_w**；或
  - 用行权 **`w_j`** 的加权 MSE 作代理指标（与 R² 不完全等价，但趋势常一致）。

---
